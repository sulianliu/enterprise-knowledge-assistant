# Enterprise Knowledge Assistant

An internal AI-powered knowledge assistant built with RAG.

## Features

- Document ingestion (Markdown, PDF)
- Chunking & embedding
- Semantic search
- LLM-based answer generation
- Source attribution

## Tech Stack

- FastAPI
- Chroma (vector DB)
- OpenAI (LLM & embeddings)

## Goal

Enable employees to query internal knowledge and get accurate, source-grounded answers.
